## Pré requisitos

- Git (https://git-scm.com)
- Node (https://nodejs.org)

## Instalando/Iniciando aplicação

- `git clone https://github.com/gabriel-me/pokedex.git`
- `cd pokedex`
- `npm install`
- `npm start`
